## 2024-05-18 - Added `aria-hidden="true"` to icon ligatures
**Learning:** The application uses Google's `material-symbols-outlined` font for icons, which relies on text ligatures (e.g., rendering "casino" as a dice icon). Screen readers read this text literally, making icon-only buttons (like those for adding/removing players) confusing if their inner text is not hidden.
**Action:** Always add `aria-hidden="true"` to decorative `<span class="material-symbols-outlined">...</span>` tags, especially when they are inside interactive elements like `<button>` that already have their own `aria-label`s.
